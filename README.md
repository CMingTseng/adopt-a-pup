# Pupadopt
Pupadopt is an Android applicaton for Puppy Adoption made by Jetpack Compose

### Screens
1. Home Screen
2. List of Puppies
3. Inquire
4. FAQ

### UI Components
1. Cards
2. LazyVerticalGrid
3. Bottom Navigation
4. Coil for Image Rendering
5. Navigation
